# stock_toss_gt

토스증권 Open API 기반 Streamlit 포트폴리오 대시보드입니다.

## 기능

- 토스증권 Open API로 내 계좌 보유 종목 조회
- API 없이 수기 포트폴리오 구성
- 종목코드, 종목명, 시장, 통화, 수량, 매입가, 현재가 직접 입력
- 국장/미장 보유 종목 분리
- 원달러 환율 표시
- 보유 종목 평가손익/수익률 계산
- 국장/미장별 보유 비중 원파이차트
- 사용자가 입력한 일수 기준 일자별 종가 선 그래프
- 국장 보유 종목 최근 3거래일 개인/외국인/기관 순매수
- CSV 다운로드

## 제거된 기능

- PER 낮은 순 TOP 5 제거

## 실행

```bash
python3 -m venv .venv
source .venv/bin/activate

python -m pip install --upgrade pip
pip install -r requirements.txt

streamlit run app.py
```

## 수기 포트폴리오 사용법

1. `종목명/종목코드 검색해서 추가`에서 종목명을 검색하거나, 표에서 직접 종목을 입력합니다.
2. 수량, 매입가, 현재가를 입력합니다.
3. `수기 포트폴리오 저장` 버튼을 누릅니다.
4. 사이드바에서 `수기만 보기` 버튼을 누릅니다.

API 키가 있는 경우:

- `수기 포트폴리오 포함`을 켜면 API 계좌 종목과 수기 입력 종목이 함께 반영됩니다.
- `수기 종목 현재가 API로 보완`을 켜면 수기 종목의 현재가를 토스 현재가 API로 덮어쓸 수 있습니다.

## 일봉 차트

사이드바의 `일봉 차트 조회 일수`에 원하는 일수를 입력하세요.

예:

- 20
- 120
- 365

종목 수가 많거나 조회 일수가 길면 API 호출량이 늘어날 수 있습니다.


## 이번 수정사항

- `KeyError: ['market_label'] not in index` 오류를 수정했습니다.
- 수기 포트폴리오 저장 시 `market_label` 컬럼이 유지되도록 수정했습니다.
- 종목명/종목코드 검색 UI를 추가했습니다.
- 국장은 `pykrx`의 종목 마스터로 종목명 검색을 지원합니다.
- 미장은 주요 미국 종목/ETF 목록 기준으로 검색을 지원합니다.


## 검색 드롭다운 개선

검색어 입력 즉시 후보 종목이 드롭다운으로 표시됩니다.

예:

- `SK` 입력 → SK하이닉스, SK스퀘어 등 후보 표시
- `삼성` 입력 → 삼성전자 등 후보 표시
- `AAPL` 입력 → Apple 표시

검색은 종목명/종목코드 기준이며, 공백이 있어도 일부 매칭되도록 처리했습니다.


## 이번 UI 개선사항

- 종목 검색에서 시장 선택 필터를 제거하고 전체 시장 기준으로 검색합니다.
- 종목 추가 시 현재가는 가능한 경우 토스 현재가 API의 실제 현재가로 입력됩니다.
- 매입가도 최초 기본값은 현재가와 동일하게 들어갑니다.
- API 키가 없거나 데모 모드인 경우 데모 가격/fallback 가격으로 채워집니다.
- 수기 포트폴리오 입력 표에서 시장 컬럼을 제거했습니다. 시장은 종목코드/통화 기준으로 자동 판별합니다.
- 보유종목 탭에 `종합` 탭을 추가했습니다.
- 국장/미장 보유 비중 차트를 보유 종목 표 아래로 이동했습니다.
- 원파이 차트 대신 더 상호작용이 좋은 트리맵과 평가금액 순위 막대차트를 추가했습니다.


## 검색 리스트 방식 개선

- 검색어 입력 즉시 후보 종목이 리스트로 아래에 펼쳐집니다.
- 드롭다운을 열 필요 없이 각 종목 오른쪽의 `추가` 버튼으로 바로 추가합니다.
- 예: `S` 입력 → SK 관련 종목 포함 후보 리스트
- 예: `SK` 입력 → SK하이닉스, SK스퀘어 등 후보 리스트
- 예: `SK스` 입력 → SK스퀘어 우선 노출


## 숨겨진 API 키 사용

`.streamlit/secrets.toml`에 아래처럼 넣으면 앱 화면에 API 키 입력창을 노출하지 않고 사용할 수 있습니다.

```toml
TOSS_DEFAULT_API_KEY = "새로_재발급받은_API_KEY"
TOSS_DEFAULT_SECRET_KEY = "새로_재발급받은_SECRET_KEY"
```

이 파일은 `.gitignore`에 포함되어 GitHub에 올라가지 않습니다.

## 종목 검색 관련

토스증권 `/api/v1/stocks`는 종목 기본정보를 제공하지만, `symbols` 파라미터가 필수라서 종목명 키워드로 전체 종목을 검색하는 API는 아닙니다.
따라서 종목명 자동검색은 `pykrx`의 국장 종목 마스터와 앱 내부 미장 종목 목록을 사용합니다.

이번 버전은 fallback 종목 목록에도 `SK스퀘어(402340)`를 추가했습니다.


## 실시간 검색 입력 개선

기존 Streamlit `text_input`은 환경에 따라 Enter를 눌러야 검색 결과가 갱신됩니다.
이번 버전은 `streamlit-keyup` 컴포넌트를 사용해서 키를 입력하는 도중에 후보 리스트가 바로 갱신됩니다.

예:

- `삼` 입력 중 → 삼성전자, 삼성전기, 삼성화재 등 표시
- `SK` 입력 중 → SK하이닉스, SK스퀘어 등 표시
- `SK스` 입력 중 → SK스퀘어 우선 표시

설치:

```bash
pip install -r requirements.txt
```
